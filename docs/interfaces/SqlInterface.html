Interface SqlInterface<ResourceTypeMap>

An interface without all the internal methods of the class.

Since typescript (disappointingly) includes protected/private methods and properites when using a class as an interface, we define this "lightweight" interface for use in function parameter definitions, so as to narrow the interface to only public methods.
